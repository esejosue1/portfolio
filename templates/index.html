RFID Door Lock System Simulator

This project consists of an RFID door lock simulator system which consists of the ability to swipe selected RFID cards within the RFID sensor, for which the information of the card is abstracted, decoded, and logically processed to grant permission to the user. When the program executes, it prompts the user 3 options to select which are swiping a new card to register into memory, swiping a registered card to check for access permission, and lastly deleting a registered card from program memory. In addition, to simulate real-life access granted, we attached two servos next to each other to simulate a mini door, in which when access is granted, both servos turn 90 degrees from each other to portray the opening of a door. Two LEDs were attached to the board, one green and one red, that light up when users get access granted or denied.
